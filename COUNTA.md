# COUNTA 
Đếm số dãy a gồm n số nguyên dương thỏa:
- 0 ≤ $a_i$ ≤ 9
- $min(a_1, a_2, . . . , a_n) = 0$
- $max(a_1, a_2, . . . , a_n) = 9$
## Dữ liệu
Gồm một dòng duy nhất chứa số nguyên dương n (1 ≤ n ≤ $10^6$ ).
## Kết quả
In ra số dãy a đồ dài n thỏa điều kiện đề bài. Vì đáp án có thể rất lớn, in đáp án modulo $10^9$ + 7.
#### INPUT
```
1
```
```
10
```
#### OUTPUT
```
0
```
```
100172994
```
